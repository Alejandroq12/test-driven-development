# ğŸ“– Test Driven Development <a name="about-project"></a>

Test-Driven Development is an approach that was adopted for a project designed to practice testing. It included three challenges: implementing a factorial method, a reverse method, and a fizzbuzz method, all of which were implemented using Ruby and RSpec.

### Key Features <a name="key-features"></a>
- **It tests reversing a word.**
- **It calculates the factorial of an integer number.**
- **It displays the words fizz, buzz, or fizzbuzz when an integer is divisible by 3, 5, or both.**

This project does not require additional dependencies, just Ruby interpreter. However, it uses RuboCop as a linter and you should set it up as well.

```sh
bundle install
```
